## `full_tool.py`

CAD 辅助拆图及数据录入工具。

### 当前限制

- 输出所使用的 Excel 模板中不可包含图形对象。
- 模板 J 列之后不可存在其他内容。
- 录入范围编号时，仅支持单个连续范围。
- 同一种 CAD 图形类别，如 `line`、`circle`、`polyline`，只能对应一种小加工类型。

### 启动方式

1. 先开启需要处理的 CAD 图面。
2. 再启动 `full_tool.py`。
3. 根据右上角浮动窗口观察当前录入进度。
4. 设置表头所在行时，请输入“数据开始行数 - 2”。

### 操作流程

1. 点选编号。

   若为范围编号，依次点选范围内的第一个编号和最后一个编号。目前范围编号仅支持单个连续范围。

2. 依次点选图面长和图面宽。

3. 若当前项目没有小加工，直接点选下一组编号。

4. 若存在小加工，点选对应的小加工图示。

   选择时需注意被点选对象的 CAD 图形类别，例如 `line`、`circle` 或 `polyline`。同一种图形类别只能对应一种小加工。

5. 选择小加工后，根据弹窗提示，为对应边选择“加工”或“不加工”。

6. 完成后重复上述步骤，继续录入下一组内容。

7. 全部录入完成后结束操作，并依照弹窗流程选择输出方式。

8. 等待程序提示输出完成。

### 双击操作

在 CAD 图面中双击后，可以选择：

- 继续当前步骤；
- 放弃当前组；
- 结束录入。

### 中断恢复

如操作过程中程序意外退出或中断，重新启动程序后，可以选择继续上一次未完成的录入。

### 关于单 Sheet／多 Sheet

此处的单 Sheet／多 Sheet 功能，仅用于决定将录入内容输出到 Excel 文件中的哪个 Sheet。

多 Sheet 模式不代表将同一次录入的内容自动拆分输出至多个 Sheet。

## `excel_transform_version3.py`

用于处理由 `full_tool.py` 输出的 Excel 文件。

### 当前功能

- 计算片数；
- 输入固定厚度；
- 计算小加工数量；
- 调换长宽；
- 计算纹路；
- 转换单个文件内的一个或多个 Sheet；
- 输出为生产单格式。

### Sheet 处理方式

- 单 Sheet：在操作界面中直接选择需要转换的 Sheet。
- 多 Sheet：选择多 Sheet 模式后，在弹窗中输入需要处理的 Sheet。

### 厚度输入

支持：

- 单一固定厚度；
- 范围厚度，例如 `30~150`。

不支持在同一次转换中分别输入多个不同的固定厚度。

### 操作流程

1. 选择需要转换的 Excel 文件。
2. 选择单 Sheet 或多 Sheet 模式。
3. 输入或选择相应的 Sheet。
4. 输入厚度。
5. 启动转换。
6. 等待小加工种类弹窗。
7. 根据实际需求选择小加工的计算单位为“米”或“个”。
8. 等待转换完成。
9. 选择输出为生产单格式。
10. 选择生产单模板和目标输出位置。
11. 等待输出完成。

## `symbol_config.json`

此文件用于保存 `full_tool.py` 所使用的小加工图形配置。

需要根据实际使用的小加工图示，手动录入对应图形数据。

如需新增或修改小加工图形类别，可能还需要同步调整 `full_tool.py` 中对应的识别和映射逻辑。
