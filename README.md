# 关于此工具说明
目前，输出使用的模板内不可有图形，J列之后的部分不能有内容。

对于表头所在行，请输入数据开始的行数-2.

## full_tool.py
先开启CAD图面再启动此工具。通过右上角的浮窗观察当前进度，浮窗不可点选【单线程，无法操作浮窗】

双击图面，可以选择继续当前步骤、放弃当前组、结束录入。

操作流程：

先点选编号【若是范围编号，点选第一个和最后一个编号。范围编号只支持单范围，如A1-5~9,A2~8-3，不支持A1~5-2~8】

依次点选图面长和图面宽。

若无小加工，直接点选下一组编号，前面选择的内容会被记录成一组。

若有小加工，点选小加工的图示。注意记录被选择的符号的类别，如line、circle、polyline，一个类别只能对应一种小加工。

选择完小加工后，根据弹窗为对应的小加工选择加工边或不加工，完成后重复前述流程。

全部完成后结束录入，选择生产单模板、文件输出位置，按照弹窗流程选择，完成后等待弹窗提示输出完成即可。

关于单sheet/多sheet：仅用于将内容输出到非第一个sheet的其他sheet，而不是用于将一次录入的内容分开输出。【例如有sheet1、2，若要将内容输出到2，选择多sheet，在下拉栏选择2】

如果意外中断了操作、程序退出，重新启动可以选择继续之前的选择【不过之前的选择不会预览】

## excel_transform_version3
此表格转换器仅针对使用full_tool工具输出的表格，用于计算片数、输入厚度【固定值】、计算小加工数量、调换长宽、计算纹路。

支持一次转换单个文件内的多个sheet。单sheet按照操作界面选择，多sheet则在点选此项后进行下一步，在弹窗内输入内容。

厚度支持单个厚度和范围厚度【如30~150】，但不支持填入不同的厚度。

开始后，等待小加工种类的弹窗，按需求选择计算单位为米/个。确认后等待转换完成。

转换完成后选择输出为生产单格式，选择你的模板和目标位置输出即可。

## symbol_config
此为full_tool的小加工图。需要手动录入对应图形的数据、并按照实际情况修改该full_tool。
